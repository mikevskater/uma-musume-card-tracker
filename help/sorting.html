Sorting System

The app features both simple column sorting and advanced multi-layer sorting for complex card analysis.

Column Sorting

Click on any sortable column header to sort by that criteria:

  • 🔤 Card Name: Alphabetical by character name
  • ⭐ Rarity: R → SR → SSR
  • 🎯 Type: Alphabetical by type
  • 📅 Release Date: Chronological order
💡 Quick Tip: Click the same column header again to reverse the sort direction (ascending ↔ descending).

Multi-Layer Sorting

For advanced analysis, use the multi-layer sort system in the sidebar:

Add Sort Layer: Click "+ Add Sort Layer" to create a new sorting criteria.
Choose Category: Select what to sort by (Effect, Level, Ownership, etc.).
Select Options: For effects and skill types, choose specific values to sort by.
Set Direction: Choose ascending (↑) or descending (↓) order.
Arrange Priority: Use ↑ and ↓ buttons to reorder sort layers. Top layers have highest priority.

Sort Categories

📊 Effect

Sort by specific effect values (Speed+, Stamina+, etc.)

📈 Level

Sort by current or effective card level

✅ Ownership

Sort by owned vs unowned status

🎲 Skill Count

Sort by number of hint or event skills

🏷️ Skill Type Count

Sort by count of specific skill types

🔄 Sort Priority: Sorts are applied in order from top to bottom. Cards equal in the first criteria are sorted by the second, and so on.

Sort Examples

  • Find Best Speed Cards: Sort by Speed+ effect (descending), then by rarity (descending)
  • Prioritize Owned Cards: Sort by ownership (descending), then by level (descending)
  • Skill Analysis: Sort by hint skill count (descending), then by specific skill types
