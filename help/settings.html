Settings

App-wide settings that persist between sessions via your browser's local storage.

Debug Logging

Debug logging outputs diagnostic messages to the browser's Developer Console (F12). Useful for troubleshooting issues or reporting bugs.

Enable Debug Logging Master toggle — when off, no debug messages are emitted
Minimum Log Level Only messages at or above this level are shown
Per-Module Toggles

Enable logging for specific modules only. The master toggle above must be off for these to take effect individually — when the master toggle is on, all modules log.

Tip: You can also control debug logging from the browser console:
_debug.enableAll() / _debug.disableAll()
_debug.setLevel('debug')
_debug.status() — print current config
Or via URL: ?debug=DeckFinder,Worker&debugLevel=debug
