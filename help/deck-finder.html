Deck Finder

Automatically search for optimized support card decks. Set your constraints, configure scoring weights, and let the finder evaluate thousands of possible deck combinations to find the best ones.

💡 Quick Start: Choose your card pool, set any constraints, then click "Find Best Deck" to start the search. Results appear ranked by score.

Card Pool

Control which cards the finder can use when building decks:

📦 Owned vs All

Search only your owned cards, or include all cards in the database. Using owned cards gives you practical results you can build immediately.

⭐ Rarity Filter

Include or exclude cards by rarity (SSR, SR, R). Most competitive decks use SSR cards only.

🏷️ Type Filter

Include or exclude specific card types (Speed, Stamina, Power, Guts, Wisdom, Friend).

Deck Type Composition

Specify how many cards of each type you want in your deck. For example, you might want exactly 2 Speed cards and 1 Friend card.

  • Count Inputs: Set the number of cards required for each type (Speed, Stamina, Power, Guts, Wisdom, Friend).
  • "At Least" Mode: When set, the finder treats the number as a minimum rather than an exact count. Setting Speed to 2 means "at least 2 Speed cards."
  • Sum Counter: Shows the total required slots. If the sum exceeds 6, the constraint is impossible and the finder will warn you.
💡 Flexible Distribution: Set counts to 0 for types you don't care about. The finder will fill those slots with whatever scores best.

Minimum Deck Effects (Thresholds)

Set minimum values for key effects that any valid deck must meet. Decks that fall below these thresholds are excluded from results.

  • Race Bonus: Minimum combined race bonus across all 6 cards.
  • Training Effectiveness: Minimum combined training effectiveness.
  • Friendship: Minimum combined friendship bonus.
  • Energy Reduction: Minimum combined energy recovery/reduction.
⚠️ Strict Constraints: Thresholds are hard cutoffs. Setting them too high may eliminate all possible decks and return no results.

Required Skills

Require that decks provide specific hint skills:

Select Skills: Use the dropdown to pick hint skills the deck must provide.
ALL Mode: The deck must contain every selected skill. Use this when you need a specific set of skills.
ANY Mode: The deck must contain at least one of the selected skills. Use this for flexibility.

Exclusions & Inclusions

Fine-tune the search by locking specific cards in or out:

🔒 Include Cards

Force specific cards into every result deck. Useful when you know certain cards must be part of your deck.

🚫 Exclude Cards

Prevent specific cards from appearing in results. Use this to avoid cards you don't want to use.

👤 Exclude Characters

Remove all cards from a specific character. Useful when a character's cards don't fit your strategy.

Scoring Weights

Weights determine how the finder scores each deck. Higher weight means that metric is prioritized more in the final score.

  • Scenario Defaults: Each scenario has preset weights optimized for its mechanics. These are loaded automatically when you select a scenario.
  • Custom Weights: Adjust individual weight sliders to match your training strategy. For example, increase Race Bonus weight if you prioritize race performance.
  • Reset: Click "Reset to Scenario Defaults" to restore the preset weights for the current scenario.
💡 Weight Strategy: If you're unsure, start with the scenario defaults. Adjust weights after reviewing initial results — increase the weight of metrics you find too low in the top results.

Understanding Results

After the search completes, results are displayed ranked by their score:

  • Score: The weighted sum of all deck metrics. Higher is better.
  • Card List: The 6 cards that make up the deck, shown with their type and rarity.
  • Key Metrics: Race Bonus, Training Effectiveness, and other important values shown at a glance.
  • Expanded View: Click a result to see the full effect breakdown, per-card contributions, and detailed scoring.

Loading to Deck Builder

Found a deck you like? Load it directly into the Deck Builder:

Click "Load Deck": Each result has a "Load Deck" button that populates the Deck Builder with those 6 cards.
Overwrites Current Deck: Loading a result replaces whatever is currently in the Deck Builder. Save your current deck first if you want to keep it.
Review & Adjust: Once loaded, use the Deck Builder to fine-tune training assignments and review the full training breakdown.
🔄 Workflow: Use the Deck Finder to discover strong combinations, then load them into the Deck Builder for detailed analysis and customization.
