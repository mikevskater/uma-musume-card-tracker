Filtering System

The filtering system allows you to narrow down the card list using various criteria. Filters are located in the left sidebar and update results in real-time.

Basic Filters

Rarity Filter: Select one or more rarities (R, SR, SSR) to show only cards of those rarities.

Ownership Filter: Show all cards, owned only, or unowned only.
Name Search: Type character names to search for specific cards.
Release Filter: Toggle to include cards not yet released in Global.

Level Display Options

Control how card levels are displayed and calculated:

  • Set All LB: Override all cards to use the same limit break level for comparison
  • Apply to Owned: Whether the global LB setting affects your owned cards
  • Show Max Potential: Display effect values at maximum level instead of current level

Advanced Filters

Click "Advanced Filters" to access more sophisticated filtering options:

  • Effect Filters: Set minimum values for specific card effects
  • Skill Filters: Filter by specific hint skills or event skills
  • Skill Type Filters: Include or exclude cards with certain skill types (pacer, debuff, etc.)
